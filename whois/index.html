What is WHOIS?

WHOIS is a public database that stores information about domain name registrations. When someone registers a domain (like example.com), they are required to provide certain details such as registrant name, contact email, and registration and expiry dates. WHOIS is like a phonebook for domain ownership.

How Does WHOIS Help?

  • Verify ownership: Check if a domain is owned and by whom.
  • Find registrar & expiry: Know when a domain expires and where it is hosted.
  • Investigate scams: Used by security researchers to trace suspicious domains.
  • Negotiate purchases: Helpful if you want to buy a domain from its current owner.

Registrar: Namecheap Inc.
Creation Date: 2010-05-10
